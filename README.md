# sistema-notas-python
Sistema de notas desenvolvido em Python para praticar lógica de programação e fundamentos de back-end.

## Funcionalidades
- Adicionar notas
- Validar valores entre 0 e 10
- Calcular média
- Exibir situação do aluno (aprovado ou reprovado)

## Tecnologias
- Python 3

## Como executar
Execute o arquivo principal:
```bash
python main.py
